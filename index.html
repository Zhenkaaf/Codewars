Чтобы группировать радио-кнопки, нужно задать им одинаковый атрибут name. Это гарантирует, что пользователь сможет выбрать только один вариант из группы. Атрибут value определяет значение, которое будет отправляться на сервер при выборе.

Выберите любимый цвет:

444
333
